# English Learn Movie Maker (ELMM)

## 概要

**English Learn Movie Maker（ELMM）** は、AIを活用して語学学習動画を自動生成し、YouTubeに公開するアプリケーションです。本アプリケーションは個人利用を目的として開発・運用されています。

日本語話者向けの英語学習、英語話者向けの日本語学習、英語話者向けのスペイン語学習、日本語話者向けの中国語学習など、複数の言語ペアに対応した学習コンテンツを毎日自動的に生成・公開しています。

## 主な機能

- **語学学習動画の自動生成**: AIが学習テーマの選定、例文の生成、画像の作成、音声合成、動画編集までを一貫して行います
- **複数言語ペア対応**: 日本語⇔英語、英語→スペイン語、日本語→中国語の学習コンテンツを提供
- **YouTubeへの自動公開**: 生成した動画を自動的にYouTubeにアップロードし、適切なタイトル・説明・サムネイルを設定します
- **分析に基づく品質改善**: YouTube Analyticsのデータを分析し、視聴者にとってより効果的なコンテンツを自動的に生成するよう改善を続けます

## 利用するサービス

本アプリケーションは以下の外部サービスと連携しています。

| サービス | 用途 |
|---------|------|
| YouTube Data API | 動画のアップロード、メタデータの設定、再生リスト管理 |
| YouTube Analytics API | 動画のパフォーマンス指標の収集・分析 |
| OpenAI API | テキストコンテンツ（テーマ、例文、説明文等）の生成 |
| Azure Speech Services | テキストから音声への合成（多言語対応） |
